# リポジトリ一覧

## 収録リポジトリ

- `typing/` 
  - `https://github.com/Nonsense001/typing`
  - 学習したHTMLのアウトプット
  - HTML / CSS / JavaScript（Vanilla JS）
  - **タイピングゲーム**。Space/ボタンで開始・停止、ストップウォッチ計測、10問クリアでタイムを**ランキング（上位5件）**に反映します。 

- `othello/`
  - `https://github.com/Nonsense001/othello`
  - 学習したPythonを用いたターミナル上でのオセロゲーム
  - Python（標準ライブラリ）
  - **CUIオセロ**。置ける場所の表示（アシストON/OFF）や、**CPU対戦（探索深さ指定）**に対応しています。

- `casino/`
  - `https://github.com/Nonsense001/casino`
  - 学習したJavascriptのアウトプット
  - HTML / CSS / JavaScript + jQuery
  - **ミニカジノ（統合ページ）**。所持金を共通化し、ミニゲーム（例：BlackJack）を選んで遊べます。所持金は**ローカル保存**されます。

- `rpg_battle/`
  - `https://github.com/Nonsense001/rpg_batte`
  - Vibe codingでの実装
  - **フロント**: Vue 3 + TypeScript + Vite + Vue Router / **バックエンド**: Node.js(Express) + TypeScript / **DB**: MySQL
  - **RPGバトルシミュレーター（Webアプリ）**。ターン制コマンドバトル、育成（経験値ストックと割り振り）、階層型ダンジョン、マスタデータ管理などを実装しています。設計・仕様ドキュメントも同梱。

- `rhythm/`
  - `https://github.com/Nonsense001/rhythm`
  - Vibe codingでの実装
  - **フロント**: React + TypeScript + Vite
  - **リズム構築アプリ（Webアプリ）**。4x4グリッドへの音符配置、**ベクトル（矢印）による再生方向の制御**、BPM変更、クリック音のON/OFFなどを実装しています。

- `react_study/`
  - `https://github.com/Nonsense001/react_study`
  - React学習のためのアプリを作る(未完成)
  - **フロント**: React + TypeScript + Vite
  - **React学習用リポジトリ**。基礎から応用（Hooks, Routing, Zustand）までを網羅する学習記録。**インプットとアウトプットを組み合わせたカリキュラム**で進行し、複数の小規模アプリや最終課題を収録予定。
